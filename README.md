# 🏠 房屋家具配置工具（room-planner）

> 純前端、零安裝的房屋平面配置工具。設定房屋公分尺寸後，拖拉家具、自由畫牆，用真實比例尺快速確認空間與格局。
>
> A dependency-free, single-page floor-plan tool. Set your room size in centimetres, then drag furniture (with real cm dimensions), draw walls freely, and verify the layout. Built with vanilla JS + SVG — the SVG `viewBox` is the real-world cm grid, so the scale is exact and automatic.

**🔗 線上使用：https://lancelotwang114.github.io/room-planner/**

開瀏覽器即用，無需後端、無相依套件、無安裝。

## 🚀 快速開始

1. 在左側「房屋尺寸」輸入你家實際的寬 × 深（公分），即時換算坪數。
2. 從「家具目錄」**拖曳**家具進畫布，選取後拖四角改大小、拖上方圓點旋轉。
3. 切「📏 畫牆」模式，按住拖曳畫出隔間牆（牆面對齊格線，家具會自動貼齊）。
4. 滿意後「💾 存檔」成自含 HTML，或「📷 複製圖」貼到 LINE／群組討論。

懶得從零開始？按頂部「📐 範例」載入一個三房兩廳格局直接改。

## ✨ 功能

- **真實比例尺**：SVG `viewBox` 直接以公分為座標，輸入房屋寬×深即定比例，家具按 cm 自動縮放，即時顯示**坪數**。
- **縮放／平移**：滾輪縮放、右下角縮放鈕、`⤢` 符合視窗；空白鍵／中鍵／✋平移模式拖曳畫布。
- **家具目錄**：床／客廳／餐廚／收納／衛浴／開口，附台灣常見預設尺寸（如雙人床 152×190 cm），**拖曳**放入畫布。支援**自訂家具**（自訂名稱＋寬深）。
- **家具編輯**：四角控制點拖曳改大小（支援任意旋轉角）＋上方旋轉控制點（吸附 15°，Shift 自由）；左側可輸入精確 cm；加標籤。
- **門開門方向**：門顯示開向與迴旋扇形，可循環 4 種方位。
- **自由畫牆＋可編輯**：拖曳畫一道牆，圍出 L 型／不規則隔間，顯示長度 cm；選取後可拖移整段、拖端點改長度。牆填滿一格、牆面對齊格線。
- **家具貼牆**：拖近牆面自動吸附貼齊（同時落在格線上）。
- **框選群組**：拖框多選、整組移動；Shift 點擊加選。
- **多分頁對比**：同一檔案多個格局分頁，可新增／複製／改名／刪除，方便比較方案。
- **格線吸附**：5／10／25／50 cm 格線，可開關吸附。
- **復原／重做**：80 步，`Ctrl+Z` / `Ctrl+Y`。
- **存檔／讀取**：存成自含 HTML（雙擊即看，含所有分頁），可讀回 `.html` / `.json` 續編；瀏覽器自動暫存。
- **複製圖**：目前分頁平面圖轉 PNG 複製到剪貼簿。

## ⌨️ 快捷鍵

| 鍵 | 動作 |
|----|------|
| 方向鍵 | 微調選取物件（Shift = 1 cm 微調） |
| `R` | 旋轉家具 90° |
| `空白鍵`＋拖曳 / 中鍵拖曳 | 平移畫布 |
| 滾輪 | 縮放 |
| `Delete` / `Backspace` | 刪除選取 |
| `Ctrl+Z` / `Ctrl+Y` | 復原 / 重做 |
| `Esc` | 取消選取 |

## 🛠 技術

純 vanilla JavaScript + SVG + CSS，無框架、無建置步驟、無相依套件。

```
index.html   結構、工具列、分頁列、目錄、SVG 畫布
app.js       狀態(多分頁)、SVG 渲染、拖拉、畫牆、家具目錄、存讀、匯出
style.css    俯視平面圖風格
```

關鍵設計：SVG `viewBox` 直接以公分為座標系，比例尺自動成立；牆填滿一格使牆面對齊格線，家具吸附格線即同時貼齊牆面。
